# Stock Market Analysis Website

This is a simple web application that provides real-time stock market data, a chatbot, and a trading simulation environment.

## How to run the website

1.  Open the `index.html` file in your web browser.
2.  The website will be displayed.

## How to get an Alpha Vantage API key

The website uses the Alpha Vantage API to get real-time stock data. You need to get a free API key from their website.

1.  Go to the [Alpha Vantage website](https://www.alphavantage.co/support/#api-key).
2.  Claim your free API key.
3.  Open the `app.js` file and replace `YOUR_API_KEY` with your own API key.

## Features

*   **Real-time stock chart:** The website displays a real-time chart for a sample stock (IBM).
*   **Chatbot:** A simple chatbot that can answer basic questions about the stock market.
*   **Trading simulation:** A simulated environment where you can buy and sell stocks with a virtual portfolio.
